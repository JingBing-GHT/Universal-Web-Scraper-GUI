# Universal-Web-Scraper-GUI
带有GUI界面的全能网页爬虫工具，支持批量URL爬取和数据导出
# 全能网页爬虫工具

![Python](https://img.shields.io/badge/Python-3.6%2B-blue)
![License](https://img.shields.io/badge/License-MIT-green)
![GUI](https://img.shields.io/badge/GUI-Tkinter-orange)

一个带有图形界面的多功能网页爬虫工具，支持单URL和批量URL爬取，可自动提取网页标题、正文内容、图片链接等信息，并导出为Excel/CSV格式。

## ✨ 功能特点

- 🖥️ **友好图形界面** - 基于Tkinter开发的直观易用的GUI
- 🔗 **灵活爬取模式** - 支持单URL和批量URL爬取
- 📊 **多数据提取** - 自动提取标题、正文预览、图片链接、页面链接
- 💾 **多种导出格式** - 支持Excel和CSV格式导出
- ⚙️ **自定义配置** - 可设置User-Agent、请求头等参数
- 🚀 **稳定高效** - 内置异常处理和进度显示

## 🛠 技术栈

- **Python 3.6+**
- **Tkinter** - 图形用户界面
- **Requests** - 网络请求
- **BeautifulSoup4** - HTML解析
- **Pandas** - 数据处理和导出
- **Openpyxl** - Excel文件操作

## 📦 安装依赖

```bash
pip install requests beautifulsoup4 pandas openpyxl
```

## 🚀 使用方法

1. **单URL爬取**：在"目标URL"输入框中输入要爬取的网址
2. **批量爬取**：在"批量URL"区域输入多个网址（每行一个）
3. **设置参数**：可自定义User-Agent（可选）
4. **开始爬取**：点击"开始爬取"按钮
5. **导出数据**：爬取完成后点击"导出数据"保存结果
